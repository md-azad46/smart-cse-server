# 🚀 Smart CSE Server

## 📌 Project Overview

**Smart CSE Server** is the backend API for the Smart CSE platform.  
It provides RESTful endpoints, authentication, database operations, file uploads, and business logic to support frontend application features like user management, attendance, results, routines, notices, and more.

This server is built using **Node.js**, **Express.js**, **MongoDB**, and other key backend tools. It handles routing, authentication, role‑based access, file storage (Cloudinary), and secure data operations.

---

## 🧠 Features

✅ User registration & login (JWT authentication)  
✅ Role‑based access (admin, teacher, student)  
✅ REST APIs for courses, attendance, results, routines, notice boards  
✅ Upload & delete images via Cloudinary  
✅ Password reset via email (NodeMailer + secure token)  
✅ Student/teacher/user analytics endpoints  
✅ Attendance and schedule APIs  
✅ Secure CORS configuration
